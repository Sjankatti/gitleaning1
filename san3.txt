1)  git --version
2) download and install git 
3) create GitHub accout 
4) configure GitHub account to a local git
   git config --global user.email "sjankatti@gmail.com"
   git config --global user.name "sjankatti"
5) create a folder of project or go to the folder by using cd path 
6) git init
7) git status
8) git add filename  or git add .
9) git commit -m "added san1.txt"
10) git remote add origin https://github.com/Sjankatti/gitleaning1.git
11)git branch -M main
12) git push -u origin master
